# Screens & UX

## User Flow

```
Home → Camera → Analyzing... → Results → Review/Correct (optional)
                                  │
                                  ▼
                               History
```

Phase 2 adds: Auth Screen, Profile Screen
Phase 3 adds: Plants List, Add/Edit Plant, Plant Detail + Timeline, Export/Report
Phase 4 adds: Lens Guide, Learn/Education

---

## Phase 1 Screens

### Screen 1: Home

- App logo/branding
- Large "Analyze" CTA button
- Recent analyses preview (thumbnails + maturity badges)
- "History" link

### Screen 2: Camera/Upload

- Expo Camera with macro photo capture
- Gallery picker for existing photos
- Tips overlay: "Hold phone 2-3cm from trichomes"

### Screen 3: Analyzing (Loading)

- Progress animation
- Status text: "Detecting trichomes...", "Classifying...", "Analyzing stigmas..."

### Screen 4: Results

- Annotated full image (trichomes colored: grey=clear, white=cloudy, orange=amber; stigma outlines)
- Trichome distribution bar (clear/cloudy/amber percentages)
- Stigma orange/green ratio
- Maturity stage badge (Early / Developing / Peak / Mature / Late)
- Harvest recommendation text
- Buttons: "Review Detections", "Save", "Share"

### Screen 5: Review/Correct

Two tabs: **Trichomes** | **Stigmas**

**Trichomes tab:**
- Grid of cropped trichome images (extracted from bbox regions)
- Each has a colored border: grey = clear, white = cloudy, orange = amber
- Tap to cycle classification (clear -> cloudy -> amber -> clear)
- Live-updating distribution stats at top

**Stigmas tab:**
- Grid of cropped stigma segments with color overlay
- Shows orange/green ratio per stigma
- Tap to toggle dominant color classification
- Live-updating overall ratio at top

"Confirm & Save" button — stores both original + corrected data

### Screen 6: History

- Scrollable list of past analyses
- Thumbnail + date + maturity badge
- Tap to view full results

## Phase 3 Screens

### Screen 9: Plants List

- List of user's plants grouped by active/harvested
- Each card: plant name, strain, last analysis thumbnail, current maturity badge
- "Add Plant" FAB button
- Tap to view plant detail

### Screen 10: Add/Edit Plant

- Form fields: name, strain (text input + common strains autocomplete), planted date
- Optional: notes field
- Save / Cancel

### Screen 11: Plant Detail + Timeline

- Plant info header (name, strain, planted date, days since planting)
- **Maturity Chart**: Area chart with time on X-axis, percentage on Y-axis
  - Three stacked areas: clear (grey), cloudy (white), amber (orange)
  - Each data point is an analysis
  - Shows trichome distribution evolution over days/weeks
- **Harvest Prediction Banner**: "Estimated 5-7 days to peak potency" (based on trend)
- **Analysis Cards Timeline**: Scrollable vertical list of analysis cards
  - Each card: thumbnail, date, distribution bar, maturity badge
  - Tap to view full results
- **Quick Analyze** button: takes a new photo linked to this plant

### Screen 12: Export/Report

- Accessible from Plant Detail screen
- Generate PDF report: plant info, maturity chart, all analyses summary
- Export as CSV: raw data for each analysis (date, trichome counts, percentages, stigma ratios)
- Share via native share sheet or email

## Phase 4 Screens

### Screen 13: Lens Guide

- Hero section: "You need a macro lens" with example comparison (phone camera vs macro lens)
- Recommended lenses by tier:
  - Budget ($10-20): Clip-on 10X macro lenses (Amazon links)
  - Mid ($20-50): Higher quality glass clip-ons
  - Best: iPhone 14 Pro+ built-in macro mode (no accessory needed)
- "How to attach" — short visual guide for clip-on lenses
- Example photos: what good macro trichome images look like
- Accessible from Home screen and Camera screen

### Screen 14: Learn / Education

- Tab in bottom navigation or accessible from Home
- Sections:
  - **Trichome Basics**: What are trichomes, why they matter for potency
  - **Color Stages**: Clear (high THC, not ready) -> Cloudy (peak THC) -> Amber (CBN, sedative)
  - **Stigma Color**: Green (immature) -> Orange (mature) and what it means
  - **Harvest Timing**: How to decide when to harvest based on desired effects
  - **Using the App**: How to take good photos, how to read results
- Content derived from published paper, simplified for growers
- Illustrated with example images from the dataset
